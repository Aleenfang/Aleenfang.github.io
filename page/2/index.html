nextTick的用法

​ nextTick是等待下一次DOM更新的工具方法。因为Vue视图是异步更新的，Vue会监听数据变化，在同一事件循环中所有的数据变化完成之后再执行视图更新。所以当我们对数据进行修改之后，可能是拿不到更新之后的DOM的，这个时候就需要借助nextTick工具方法来确保我们拿到的是更新之后的DOM。

nextTick使用场景

  1. 在created生命周期进行dom操作的时候（比如说使用echarts时需要绑定dom）
  2. 在数据变化的时候，需要使用数据变化之后的DOM

script标签

将JavaScript引入HTML的方式主要是使用script标签。script标签可以放在>标签里面，放在这里除非使用async、defer属性，不然都无法访问到dom对象，因为执行它里面的代码的时候dom元素还没有挂载，也可以放在标签里面，除非使用async、defer属性，不然放在这里只可以拿到script标签前面的dom元素，拿不到script标签后面的dom元素，因为执行的时候dom元素还没有挂载。如果不给script标签添加async属性或者defer属性，浏览器会同步下载JavaScript代码并同步执行，这个时候会阻塞浏览器的其他动作。

script标签的属性

  1. async

    表示应该立即下载脚本，但是异步执行脚本里面的内容。也就是说不会阻碍HTML其他的正常动作执行。只对外部引入脚本有效。当有多个async时，浏览器不能保证是否按照先后顺序执行。

  2. defer

    浏览器对这个属性的script标签的处理方式类似，区别是：浏览器会按照标签出现的先后次序执行脚本。

  3. crosorigin

    用于配置相关请求CORS(跨域资源共享)设置。添加crosorigin之后，控制台会显示一个比较详细的报错信息。

  4. src

    用户引入外部的js文件，src属性的值是一个url，当浏览器解析到这个地方的时候，会向该url发送一个get请求，这个get请求不受同源策略的限制，但是下载过来的js代码再发送网络请求的时候又会受到同源策略的限制。由script标签的get请求不受同源策略限制的特性，衍生出了JSONP这种解决跨域问题的方法。当为script标签设置了src属性之后，浏览器就会只执行外部引入的JavaScript，忽略script标签内部的行内JavaScript。

  5. type

    当将type的值设置为module时，浏览器就会将该script标签视为一个模块，从而可以使用es6的import语法导入模块和export语法导出模块。

防抖函数(debounce)的实现

防抖函数：就是当一个函数可能会在一个指定时间间隔(t1)内重复调用时，防抖函数能让这个函数不执行，当距离上次调用这个函数的时间间隔(t2)超过指定的时间间隔(t1)还没有再次调用这个函数时，这个函数就会执行。

举例：

在这里，如果用户一直重复点击这个按钮的话，就会在控制台中一直输出 “Clicked !” 。

​ 有的时候，我们不希望用户每点击一次就执行一次事件处理函数，而是不再点击之后执行一次这个函数。

​ 防抖函数有两个参数：需要添加防抖功能的函数，延迟的时间

​ 防抖函数需要传入参数，当给防抖函数传入参数的时候就避免不了防抖函数的立即执行，但是我们不希望在传入参数的时候防抖函数就立即执行，所以就返回一个函数，这样当给防抖函数传入参数的时候，就可以不立即执行我们传进去的函数了。在自己内部返回另外一个函数的又叫高阶函数

​

​ 由于需要在指定时间间隔之后才执行这个函数，所以我们第一时间想到的就是使用定时器

​ 当函数重复执行的时候，我们希望取消这个执行的操作，于是就想到了清除这个定时器

​ 由于我们需要清除的是上次生成的定时器，所以timer的定义就不能放在返回的函数里面，这样存储定时器ID的timer变量在定义防抖函数的时候就声明了，然后每次清除的timer就是上次执行的防抖函数生成的定时器。

css实现元素水平和垂直居中的方法

一、使用absolute 绝对定位

  1. 给父元素设置相对定位 relative

  2. 给子元素设置绝对定位 absolute的时候就会以父元素为参照对象。

  3. 给子元素的left和top设置为50%，由于绝对定位的top和left是子盒子外边框与父盒子的外边框的距离，但是这样设置之后，所以这个时候子盒子还没有居中。

  4. 为了解决这个问题，子盒子设置transform:translate(-50%,-50%),translate中的百分比是相对于盒子本身的宽和高的，就相当于把多移动的这部分补回去了，这就实现了垂直水平居中。

二、使用flex 弹性盒子布局

  1. 给父元素设置 display:flex;

  2. 给父元素设置 justify-content:center; 实现水平居中

  3. 给父元素设置 align-items:center; 实现垂直居中

三、使用 grid 栅格布局

  1. 给父元素设置 display：grid;

  2. 给父元素设置place-items:center;

    place-items 是 align-items 和 justify-items的简写

    place-content 是align-content 和 justify-content的简写

使用table

  1. 给父盒子设置display:table;

  2. 给子盒子设置 display:table-cell;

  3. 给子盒子设置 vertical-align: middle; 实现垂直居中

    vertical-align: 是设置基线

  4. 给子盒子设置 text-align:center;

    text-align:可以给内联元素设置水平居中
