vue生命周期

发表于 2022-07-03 更新于 2022-08-05

声明周期

  • beforeCreate
  • created
  • beforeMount
  • Mounted
  • beforeUpdate
  • updated
  • activated
  • deactivated
  • beforedestroy（在vue3中为beforeUnmounted）
  • destroyed（在vue3中为unmounted）

一旦进入页面或者组件，就会执行这些生命周期

  1. beforeCreate 这个地方啥也没有
  2. created 这个地方有data，无$el
  3. beforeMount 这个地方有data,无$el
  4. mounted 这个电放有data，有$el

如果加入了keep-alive 会多两个生命周期

  • activated //进入
  • deactivated //离开

如果加入了keep-alive，生命周期情况

  1. beforeCreated

  2. created

  3. beforeMounted

  4. mounted

  5. activated 加入keep-alive组件

当离开keep-alive组件的时候

  • deactivated

当再次进入keep-alive组件的时候

  • avtivated 当离开了keep-alive组件，并再次进入的时候，就不会执行之前的那四个生命周期了，只会执行activated
