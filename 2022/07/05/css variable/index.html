CSS variable

variable 也就是 css 变量，不同于less，sass等预处理器，这个是原生css支持的

优点：

可以减少样式的重复定义

缺点：

不兼容旧的浏览器

使用方法：

  1. –variable:

  2. : var(–variable)

注意事项：

  1. css变量名称以 “ – ” 开头 ，以 “ : “ 进行赋值符号，

  2. css变量要在定义选择器之后再定义。
